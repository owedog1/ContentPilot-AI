ANALYTICS AGENT

   └─ Process: Calculate KPIs, identify trends, generate reports
   └─ Output: Reports, recommendations, dashboard data, alerts

BILLING AGENT
└─ Input: Payment events, subscription changes, refund requests

   └─ Output: Combined results, health status, execution logs
